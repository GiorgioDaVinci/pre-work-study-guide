    GIT

  • Git status - this checks what branch we are currently on.
  • pwd - displays the current working directory.
  • ls - displays the files in the current working directory.
  • cd - this changes directory.
  • cd .. - this navigates backwards one at a time to the previous directory.
  • mkdir - this makes a new directory.
  • touch (name of file).txt - makes a new text file within a directory.
  • start (name of file).txt - opens the file in the directory.
  • git checkout -b fetaure/starter-code(name of branch) : this creates a new branch separate from the main branch.
  • -b : creates a new branch.
  • feauture/starter-code: this represents the name of the new branch.
  • git add -A: this indicates that changes from the staging area are to be added in the next commit.
  • git pull origin main: this ensures the code you are working is the updated version.
  • git push origin feature/starter-code: this pushes the updated version of the code to the repository.
